CAUHOI

Cho xâu st có độ dài tối đa là 255 kí tự.

Hãy cho biết số lượng chữ số \'0\', số chữ số \'1\', ..., số chữ số
\'9\' có trong xâu st đó.

**Dữ liệu:** Cho trong tệp BAI3.INP: là một xâu độ dài tối đa 255 kí tự

**Kết quả:** Ghi ra tệp BAI3.OUT: gồm một dòng chứa 10 số nguyên, trong
đó số thứ i cho biết là m lần xuất hiện của chữ số i - 1 trong xâu st.

**Ví dụ:**

+---------------+----------------+-------------------------------------+
| BAI3.INP      | BAI3.OUT       | Giải thích                          |
+===============+================+=====================================+
| > 111222255   | 0 3 4 0 0 2 0  | Số thứ nhất là 0 cho biết số 0 xuất |
|               | 0 0 0          | hiện 1 lần                          |
|               |                |                                     |
|               |                | Số thứ hai là 3 cho biết số 1 xuất  |
|               |                | hiện 3 lần                          |
|               |                |                                     |
|               |                | Số thứ ba là 4 cho biết số 2 xuất   |
|               |                | hiện 4 lần ...                      |
+---------------+----------------+-------------------------------------+

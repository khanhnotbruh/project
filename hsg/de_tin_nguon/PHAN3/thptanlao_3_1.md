CAUHOI

**Bài 3: Dãy con**

Cho dãy gồm n các số nguyên a~1~, a~2~, \..., a~n~ và số nguyên S.

**Yêu cầu**: Hãy tìm dãy con liên tiếp a~i~, a~i+1~,\..., a~j~ (1 ≤ i ≤
j ≤ n) có tổng bằng S.

**Dữ liệu:** Vào từ file văn bản DAYCON.INP gồm:

-   Dòng đầu tiên chứa hai số nguyên n, S (1 ≤ n ≤ 10^6^; \|S\| ≤
    10^9^);

-   Trong n dòng tiếp theo, dòng thứ i ghi số nguyên a~i~ (\|a~i~\| ≤
    10^6^).

Các số trên cùng dòng viết cách nhau một dấu cách.

**Kết quả:** Ghi ra file văn bản DAYCON.OUT dòng chữ YES hoặc NO tương
ứng với tìm được hay không tìm được dãy con thỏa yêu cầu. Nếu dòng đầu
ghi YES thì dòng thứ hai ghi cặp số theo dạng (i .. j) sao cho a~i~ +
a~i+1~ + \... + a~j~ = S.

Ví dụ:

+--------------------------------+-------------------------------------+
| DAYCON.INP                     | DAYCON.OUT                          |
+================================+=====================================+
| 7 5                            | YES                                 |
|                                |                                     |
| 6                              | 2 .. 5                              |
|                                |                                     |
| -3                             |                                     |
|                                |                                     |
| 1                              |                                     |
|                                |                                     |
| 8                              |                                     |
|                                |                                     |
| -1                             |                                     |
|                                |                                     |
| 7                              |                                     |
|                                |                                     |
| -7                             |                                     |
+--------------------------------+-------------------------------------+
| 2 10                           | NO                                  |
|                                |                                     |
| -2                             |                                     |
|                                |                                     |
| 3                              |                                     |
+--------------------------------+-------------------------------------+

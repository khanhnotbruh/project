**Bài 3: Dãy con liên tiếp có tổng bằng k**

Bạn được cho biết số *N* và 1 số *K* và dãy số nguyên dương
![](media/image1.png){width="1.2513888888888889in"
height="0.19027777777777777in"}![](media/image1.png){width="1.2513888888888889in"
height="0.19027777777777777in"}. Bạn hãy tìm xem có dãy con liên tiếp
nào có tổng bằng k hay không.

**Dữ liệu:** Vào từ file văn bản **BAI3.INP**

-   Dòng 1: Chứa hai số nguyên dương N

-   N Dòng sau. Mỗi dòng chứa 1 số . Dòng thứ i chứa số A\[i\].
    (0\<A\[i\] \<1000)

**Kết quả:** Ghi ra file văn bản **BAI3.OUT**

-   Gồm 1 dòng duy nhất. Nếu tồn tại dãy con thỏa mãn thì ghi TRUE ngược
    lại ghi FALSE.

**Ví dụ:**

+---------------------+---------+-------------------------------------+
| **BAI3.INP**        | **BAI   | Giải thích                          |
|                     | 3.OUT** |                                     |
+=====================+=========+=====================================+
| 10 8                | TRUE    | Dãy con từ vị trí 3 tới 4 có tổng   |
|                     |         | là 8 , có 2 phần tử (5,3)           |
| 5 6 5 3 2 1 5 5 7 4 |         |                                     |
+---------------------+---------+-------------------------------------+

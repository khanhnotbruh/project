# **TRIPLEINC.CPP**

Cho 3 dãy số $A,B$ và $C$ đều có $N$ phần tử. Hãy đếm xem có bao nhiêu
cách chọn ra bộ ba chỉ số $(i,j,k)$ sao cho $A_{i} < B_{j} < C_{k}$

**Dữ liệu:**

-   Dòng 1 chứa hai số nguyên dương $N$

-   Dòng 2: $N$ số nguyên dương A

-   Dòng 3: $N$ số nguyên dương $B$

-   Dòng 4: $N$ số nguyên dương C

**Kết quả:** Ghi một số nguyên là đáp án

**Ví dụ:**

+----------------------+---------------------+-------------------------+
| **INPUT**            | **OUTPUT**          |                         |
+======================+=====================+=========================+
| > 3                  | > 27                | > Mọi bộ $(i,j,k)$ đều  |
| >                    |                     | > thỏa mãn              |
| > 1 1 1              |                     |                         |
| >                    |                     |                         |
| > 2 2 2              |                     |                         |
| >                    |                     |                         |
| > 3 3 3              |                     |                         |
+----------------------+---------------------+-------------------------+
| > 2                  | > 3                 | > $$(i,j,k) = (1        |
| >                    |                     | ,1,1);(1,1,2);(1,2,2)$$ |
| > 1 5                |                     |                         |
| >                    |                     |                         |
| > 2 4                |                     |                         |
| >                    |                     |                         |
| > 3 6                |                     |                         |
+----------------------+---------------------+-------------------------+
| > 5                  | > 26                |                         |
| >                    |                     |                         |
| > 12 10 1 5 3        |                     |                         |
| >                    |                     |                         |
| > 2 5 100 78 2       |                     |                         |
| >                    |                     |                         |
| > 1 6 10 80 99       |                     |                         |
+----------------------+---------------------+-------------------------+

**Chú ý:**

-   Trong tất cả các test $1 \leq A_{i},B_{i},C_{i} \leq 10^{9}$

-   50% số test có $1 \leq N \leq 1\ 000$

-   50% số test có $N \leq 10^{5}$

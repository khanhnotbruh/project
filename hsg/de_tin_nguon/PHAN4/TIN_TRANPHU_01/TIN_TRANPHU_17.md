# **RBPOINT2.CPP**

Trên trục tọa độ $Oxy$ có$\ n\ $ điểm xanh và $n$ điểm đỏ. Điểm xanh thứ
$i$ có tọa độ $b_{i}$, điểm đỏ thứ $i$ có tọa độ $r_{i}$. Với hai điểm
có tọa độ $x_{1},x_{2}$, ta định nghĩa khoảng cách giữa hai điểm đó là
$\left| x_{2} - x_{1} \right|$

Hãy tìm khoảng cách nhỏ nhất giữa một cặp điểm xanh và đỏ bất kỳ trong
số các điểm đã cho.

**Dữ liệu:**

-   Dòng 1 chứa hai số nguyên dương $n\ (1 \leq n \leq 10^{5})$

-   Dòng 2: $n$ số nguyên dương
    $b_{1},b_{2},\ldots,b_{n}\ (1 \leq b_{i} \leq 10^{9})$

-   Dòng 3: $n$ số nguyên dương
    $r_{1},r_{2},..,r_{n}\ (1 \leq r_{i} \leq 10^{9})$

**Kết quả:** Ghi một số nguyên là đáp án

| **INPUT**                         | **OUTPUT**                       |
+===================================+==================================+
| > 1                               | > 4                              |

| > 2                               |                                  |
| >                                 |                                  |
| > 6                               |                                  |
+-----------------------------------+----------------------------------+
| > 2                               | > 2                              |
| >                                 |                                  |
| > 1 7                             |                                  |
| >                                 |                                  |
| > 10 5                            |                                  |
+-----------------------------------+----------------------------------+

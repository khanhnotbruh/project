**Đỗ Văn Hồng -- THPT Lê Quý Đôn**

**Phần 2.2:**

Khi xoay ngược trang giấy 180^0^ thì các số 1, 8, 0 không thay đổi; số 6
thành số 9 và số 9 thành số 6. Các số còn lại trở nên vô nghĩa.

Các số mà khi xoay ngược tờ giấy 180^0^ không đổi thì gọi là số không
thay đổi. Ví dụ 69 khi ta xoay 180^0^ vẫn là 69.

Cho trước 1 số n hãy tìm tất cả các số bất biến có n chữ số (0\<n\<16);

**Dữ liệu**: Vào từ tệp văn bản SOKHONGDOI.INP: Ghi số n.

**Kết quả**: Ghi ra tệp văn bản SOKHONGDOI.OUT: Ghi các số không thay
đổi tìm được mỗi số 1 dòng.

**Ví dụ**:

+-----------------------------------+-----------------------------------+
| **SOKHONGDOI.INP**                | **SOKHONGDOI.OUT**                |
+===================================+===================================+
| 2                                 | 11                                |
|                                   |                                   |
|                                   | 69                                |
|                                   |                                   |
|                                   | 88                                |
|                                   |                                   |
|                                   | 96                                |
+-----------------------------------+-----------------------------------+

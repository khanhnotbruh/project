**Bài 2: Tìm số**

Khi viết các số tự nhiên tăng dần từ 1, 2, 3,... liên tiếp nhau, ta nhận
được một dãy các chữ số thập phân vô hạn, ví dụ:
1234567891011121314151617181920...

***[Yêu cầu]{.underline}*:** Hãy tìm chữ số thứ N của dãy số vô hạn
trên.

***[Dữ liệu vào:]{.underline}*** từ file 'bai1n.inp' gồm một số dòng,
mỗi dòng ghi một số nguyên dương N (N\<10^9^).

***[Kết quả ra:]{.underline}*** file ' bai1n.out', với mỗi số N đọc được
từ file bai1n.inp, ghi trên dòng tương ứng chữ số thứ N của dãy.

***[Ví dụ]{.underline}:***

+-----------------------------------+----------------------------------+
| BAI1.INT                          | BAI1.OUT                         |
+-----------------------------------+----------------------------------+
| 5                                 | 5                                |
|                                   |                                  |
| 10                                | 1                                |
|                                   |                                  |
| 54                                | 3                                |
+-----------------------------------+----------------------------------+

**CAUHOI**

**Bài 2.**

Cho một dãy N số nguyên $a_{1},a_{2},a_{3},\ldots,a_{N}$
$\left( \left| a_{i} \right| \leq 1000,\ \ \forall i = 1..N \right)$.

**Yêu cầu:**

a)  Tìm giá trị lớn nhất Max và nhỏ nhất Min trong dãy số nguyên đã cho?

b)  Tìm trong dãy đã cho phần tử $a_{i}$ sao cho biểu thức
    $\left( Max - a_{i} \right).\left( a_{i} - Min \right)$ đạt giá trị
    lớn nhất trong đó Max và Min tương ứng là giá trị lớn nhất và nhỏ
    nhất ở câu (a).

**Dữ liệu vào:** Cho trong tệp BAI2.INP

-   Dòng 1: Số nguyên dương N $(3 \leq N \leq 1000)$ là số lượng phần tử
    của dãy số.

-   Dòng 2: Ghi giá trị các phần tử của dãy số. Các phần tử được ghi
    cách nhau bởi ít nhất một dấu cách trống

**Kết quả:** Ghi ra tệp BAI2.OUT:

-   Dòng 1: Giá trị Max

-   Dòng 2: Giá trị Min

-   Dòng 3: Giá trị $a_{i}$ tìm được sao cho biểu thức
    $\left( Max - a_{i} \right).\left( a_{i} - Min \right)$ đạt giá trị
    lớn nhất. Nếu có nhiều giá trị $a_{i}$ thỏa mãn thì ghi giá trị có
    chỉ số i nhỏ nhất.

**Ví dụ:**

+----------------+-----------------+----------------------------------+
| **BAI2.INP**   | **BAI2.OUT**    | **Giải thích**                   |
+================+=================+==================================+
| 6              | 7               | Max = 7                          |
|                |                 |                                  |
| 5 1 3 2 7 5    | 1               | Min = 1                          |
|                |                 |                                  |
|                | 5               | Có 3 phần tử của dãy số thỏa mãn |
|                |                 | biểu thức đã cho đạt giá trị lớn |
|                |                 | nhất là                          |
|                |                 | $a_{1}                           |
|                |                 |  = 5;\ a_{3} = 3{;\ a}_{6} = 5$. |
|                |                 | Theo yêu cầu đề bài ta in ra giá |
|                |                 | trị $a_{1}$.                     |
+----------------+-----------------+----------------------------------+

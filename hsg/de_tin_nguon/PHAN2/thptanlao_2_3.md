CAUHOI

**Bài 2 (7 điểm):**

Có 2 số nguyên tố $a$ và $b$ với $a\  \neq \ b;\ b\  \neq \ 2$ và
$b\  \neq \ 5$. Tý thực hiện chia a : b thì được một số x là thập phân
vô hạn tuần hoàn.

**Yêu cầu:** Cho trước số nguyên dương n (n ≤ 10^16^). Hãy tìm chữ số
thứ n sau dấu phẩy của số x.

**Dữ liệu vào:** Đọc ở file văn bản **C2SNT.INP** 3 số a, b, n. Trong
đó:

-   Dòng thứ nhất chứa số nguyên tố a (a ≤ $10^{7}$).

-   Dòng thứ hai chứa số nguyên tố b (b ≠ a; b ≠ 2; b ≠ 5; b ≤
    $10^{7}$).

-   Dòng thứ ba chứa số nguyên dương n (n ≤ 10^16^).

**Dữ liệu ra:** Ghi ra File văn bản **C2SNT.OUT** một chữ số thứ n sau
dấu phẩy của số x.

Ví dụ:

+-----------------------------------+-----------------------------------+
| **C2SNT.INP**                     | **C2SNT.OUT**                     |
+===================================+===================================+
| 5                                 | 4                                 |
|                                   |                                   |
| 7                                 |                                   |
|                                   |                                   |
| 15                                |                                   |
+-----------------------------------+-----------------------------------+

[**Giải thích**:]{.underline} *x = 5 : 7 = 0,71428571428571**4**285...
Chữ số thứ 15 sau dấu phẩy của số x là chữ số 4.*
